## 4. Technical SEO Standards

Title length: Aim for 50-60 characters. Put the primary keyword and brand near the front where natural.

Meta descriptions: Aim for 140-160 characters. Include the value proposition, location where relevant and a natural CTA.

Headings:

- Use one H1 only per page.
- Use H2s for major sections.
- Use H3s for supporting subsections, cards, FAQs and process steps.
- Keep heading hierarchy logical and avoid skipping levels where possible.

Images:

- Use descriptive image alt text that explains the subject and context.
- Avoid keyword stuffing.
- Decorative images can use empty alt text.
- Client/video thumbnails should identify the client or video type where appropriate.

Canonical URLs:

- Each indexable page should have a self-referencing canonical URL.
- Avoid multiple URL versions such as `/portfolio` and `/portfolio.html` competing.

Open Graph tags:

- Add `og:title`.
- Add `og:description`.
- Add `og:image`.
- Add `og:url`.
- Add `og:type`.

Twitter Card tags:

- Use `twitter:card` set to `summary_large_image`.
- Add `twitter:title`.
- Add `twitter:description`.
- Add `twitter:image`.

JSON-LD structured data:

- Add Organization or LocalBusiness schema site-wide.
- Add Service schema to service pages.
- Add VideoObject schema where videos are embedded or featured.
- Add FAQPage schema where FAQs exist.
- Add BreadcrumbList schema on all key inner pages.

Sitemap:

- Maintain an XML sitemap including all indexable pages.
- Exclude thank-you pages, private funnel pages and utility pages where appropriate.
- Submit sitemap in Google Search Console.

robots.txt:

- Allow crawling of public pages and assets.
- Reference the XML sitemap.
- Disallow private, duplicate or non-indexable funnel utility pages if required.
